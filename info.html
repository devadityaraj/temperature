"Temperature Monitoring System"By Aditya Raj

This HTML page gets the realtime Temperature and the humidity data from "DHT 11" sensor in room using Firebase. The AQI Index, Wind Speed, Pressure data is of the city "Greater Noida" and can be only changed by the changing the city name in the script file of the code in the GitHub.

Temperature Data

Temperature & Humidity Data is shown by the current physical presense of the Sensor and based on DHT 11 Sensor. It is "NOT FOR OUTDOORS" as it shows temperature (in room) where setup is present.

Wind Speed

This data is given by the "Open Meteo" for the city Greater Noida. The city can be only changed by the changing the JavaScript file of the WebPage. It maybe not accurate and is only for "OUTDOORS".

AQI Data

This data is given by the "WAQI" for the city "Greater Noida" and can be only changed by the changing the JavaScript file. It maybe not accurate and is only for "OUTDOORS".
